## viaclient

1.12.2 u3 eaglercraft client with viaversion, viabackwards, and several of its dependencies ported to it with the goal of having a seamless experience on more recent servers.

what this client is:
- 1.12.2 eaglercraft
- client side viaversion port
- backports newer client versioned features/blocks/items/entities (only 1.13.2 so far)
- via features only works server side
- normal minecraft client with some modules

what this client isnt: 
- 1.21 eaglercraft
- has support for 1.18.2 and up
- implements all features from all versions
- requires a plugin
- is a singleplayer feature backport

what this client is trying to achieve:
- porting all 1.18.2 features
